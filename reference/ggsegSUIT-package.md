# ggsegSUIT: SUIT Cerebellar Lobular Atlas for the 'ggseg' Ecosystem

SUIT cerebellar parcellation into anatomical lobules and deep nuclei for
the 'ggseg' ecosystem. Provides a unified 'ggseg_atlas' object with both
2D polygon geometry and 3D vertex indices, for use with 'ggseg' and
'ggseg3d'.
