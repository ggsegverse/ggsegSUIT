# ggsegSUIT 2.0.0

* Converted to cerebellar atlas type using SUIT flatmap projection
* Proper anatomical labels with hemisphere prefixes (left_, right_, vermis_)
* Deep cerebellar nuclei (Dentate, Interposed, Fastigial) as separate 3D meshes
* Removed lazy-loaded data; atlas accessed via `suit()` function

# ggsegSUIT 1.0.0

* Initial release with `suit` cerebellar lobular atlas as a unified
  `ggseg_atlas` object for the ggseg 2.0 ecosystem.
