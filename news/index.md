# Changelog

## ggsegSUIT 0.0.1

- Cerebellar atlas type using SUIT flatmap projection
- Anatomical labels with hemisphere prefixes (left\_, right\_, vermis\_)
- Deep cerebellar nuclei (Dentate, Interposed, Fastigial) as separate 3D
  meshes
- Atlas accessed via
  [`suit()`](https://ggseg.github.io/ggsegSUIT/reference/suit.md)
  function
